# CERG Landing — Контекст проекта

## Описание проекта

**ЦЕРГ (Центр Регрессивного Гипноза)** — лендинг для центра, специализирующегося на гипнотерапии, регрессивной терапии и обучении специалистов методам гипноза.

### Цели проекта
- Привлечение клиентов на консультации и сеансы гипнотерапии
- Продвижение образовательных программ по гипнозу
- Сбор заявок через формы обратной связи
- Интеграция с CRM для управления лидами
- Управление контентом через CMS (без привлечения разработчика)

### Целевая аудитория
- Люди с психологическими проблемами (тревожность, фобии, зависимости)
- Психологи, психотерапевты, врачи — для обучения
- Коучи и консультанты

---

## Технологический стек

### Frontend
- **Framework:** Astro 4+
- **Язык:** TypeScript
- **Стили:** Tailwind CSS + CSS Variables
- **Анимации:** CSS Animations + View Transitions API
- **Компоненты:** Astro Components (.astro)
- **Интерактивные острова:** Preact (для форм, каруселей)

### CMS
- **Decap CMS** (ex-Netlify CMS)
  - Git-based CMS (хранение в репозитории)
  - Бесплатно навсегда
  - Визуальный редактор контента
  - Не требует отдельного сервера

### Интеграции
- **CRM:** AmoCRM / Bitrix24 (через webhook/API)
- **Мессенджеры:** 
  - Telegram Bot API (уведомления о заявках)
  - WhatsApp Business API
  - VK API
- **Email:** Formspree / Web3Forms (бесплатные планы)
- **Аналитика:** Яндекс.Метрика + Google Analytics 4

### Инфраструктура
- **Хостинг (основной):** VPS в РФ (Timeweb / REG.RU / Selectel)
- **Хостинг (альтернатива):** Cloudflare Pages (бесплатно, превью PR)
- **Reverse Proxy:** Nginx
- **SSL:** Let's Encrypt (Certbot)
- **CI/CD:** GitHub Actions
- **Домен:** Российский регистратор (.ru)

### Почему Astro?
- ✅ Статический вывод — хостинг на любом сервере
- ✅ Минимум JS на клиенте (0 KB по умолчанию)
- ✅ Отличный SEO из коробки
- ✅ Простой деплой (просто папка с файлами)
- ✅ Дешёвый хостинг (от 150₽/мес)
- ✅ Долгосрочная поддержка (стабильный API)

## Принципы дизайна

### 1. Тёплая, успокаивающая эстетика
Дизайн использует мягкие бежевые и зелёные оттенки, создающие атмосферу спокойствия и доверия — критично важно для сферы психотерапии.

### 2. Визуальный storytelling
Уникальные 3D-иллюстрации с «мягкими» объектами (подушки, текстильные формы) — метафора мягкости подхода и работы с подсознанием.

### 3. Чёткая иерархия
- Крупные заголовки (uppercase) для секций
- Контрастные CTA-кнопки
- Консистентные карточки

### 4. Mobile-first адаптивность
Дизайн должен корректно масштабироваться на мобильные устройства с сохранением читаемости и удобства навигации.

### 5. Доступность (a11y)
- Контрастные цвета текста
- Семантическая разметка
- Поддержка клавиатурной навигации
- Alt-тексты для изображений

## Decap CMS

### Настройка
CMS располагается в `/public/admin/` и редактирует контент напрямую в Git-репозитории.

### Редактируемый контент
| Коллекция | Путь | Описание |
|-----------|------|----------|
| services | `src/content/services/` | Услуги |
| courses | `src/content/courses/` | Курсы |
| experts | `src/content/experts/` | Эксперты |
| reviews | `src/content/reviews/` | Отзывы |
| faq | `src/content/faq/` | Вопросы и ответы |
| settings | `src/content/settings/` | Контакты, соцсети |

### Доступ
- URL: `https://site.ru/admin/`
- Авторизация через GitHub OAuth
